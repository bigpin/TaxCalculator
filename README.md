# 日常万能助手

一个微信小程序工具集合，提供多种实用工具，包括：

## 功能模块

### 财务工具
- **个税计算器**：计算个人所得税，支持多个月份累计计算
- **养老金计算器**：计算养老金相关金额
- **汇率转换**：实时查询和转换多种货币汇率

### 更多工具
持续更新中...

## 技术栈

- 微信小程序原生框架
- TDesign 组件库
- ES6+ 语法

## 项目结构

```
pages/
  ├── index/              # 工具列表首页
  ├── tools/              # 工具页面目录
  │   ├── tax-calculator/      # 个税计算器
  │   ├── pension-calculator/  # 养老金计算器
  │   └── currency-exchange/   # 汇率转换
  └── detail/             # 详情页
```
